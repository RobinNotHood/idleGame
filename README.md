# Milo's Cozy Adventures 🏕️

A relaxing idle/incremental game where you help Milo, a friendly dog, live his best cozy life through hobbies, upgrades, and adventures!

## About the Game

**Milo's Cozy Adventures** is a character-driven idle game with a warm, optimistic tone. Guide Milo through his daily activities, upgrade his living space, develop his hobbies, and send him on adventures to collect memories.

### Core Features

- **Passive Resource Generation**: Earn "Warmth" automatically, even when offline (up to 24 hours)
- **Active Clicking**: Click on Milo to generate "Inspiration" through his hobbies
- **Living Space Upgrades**: Purchase cozy items like plants, rugs, lamps, and fireplaces to increase Warmth generation
- **Skill Development**: Improve Milo's guitar playing, reading speed, and focus to generate more Inspiration
- **Adventures (Prestige System)**: Send Milo on adventures to reset progress and earn "Memories"
- **Scrapbook of Memories**: Spend Memories on permanent upgrades that carry across all playthroughs
- **Travel Journal**: Read Milo's journal entries about his adventures
- **Item Scrapbook**: Unlock flavor text stories about items you purchase

### Game Systems

#### Resources

1. **Warmth** (Passive Currency)
   - Generates automatically over time
   - Used to purchase living space upgrades
   - Continues to generate offline

2. **Inspiration** (Active Currency)
   - Generated by clicking on Milo
   - Used to purchase skill upgrades
   - Increases with skill improvements

3. **Memories** (Prestige Currency)
   - Earned by completing Adventures
   - Used for permanent upgrades in the Scrapbook of Memories
   - Never resets

#### Progression

The game follows a prestige loop:
1. Generate Warmth and Inspiration
2. Purchase upgrades to increase generation rates
3. When ready, embark on an Adventure
4. Adventure resets Warmth and Inspiration but grants Memories
5. Spend Memories on permanent bonuses
6. Start a new cycle with permanent advantages

## How to Play

### Playing the Game

1. **Home Screen**: Click on Milo to generate Inspiration. Watch your Warmth accumulate passively.
2. **Upgrades Screen**: Purchase living space improvements and skill upgrades
3. **Memories Screen**: Spend Memories on permanent bonuses (requires at least one Adventure)
4. **Adventure Screen**: Plan and begin your next adventure when ready to prestige
5. **Journal Screen**: Read about Milo's past adventures
6. **Scrapbook Screen**: Read stories about items you've purchased

### Tips

- Your first adventure should happen when you've accumulated several hundred Warmth and Inspiration
- Memory upgrades are permanent and multiplicative - they make each subsequent run faster
- The game calculates up to 24 hours of offline Warmth generation
- Don't forget to check the Journal after each adventure for new entries!

## Installation & Running

### Option 1: Run the Executable (Easiest)

1. Download `MilosCozyAdventures.exe`
2. Double-click to run
3. The game will create a save file (`milo_save.json`) in the same directory

### Option 2: Run from Source

#### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

#### Steps

1. Clone or download this repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the game:
   ```bash
   python main.py
   ```

## Building from Source

To create your own Windows executable:

### On Windows:

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the build script:
   ```bash
   python build.py
   ```

3. The executable will be created in the `dist/` folder

### Manual Build with PyInstaller:

```bash
pyinstaller build.spec
```

## Game Design Philosophy

**Milo's Cozy Adventures** follows these core principles:

1. **Low Stress**: No timers, no pressure, no failure states
2. **Meaningful Choices**: Every upgrade feels impactful
3. **Positive Narrative**: All content is optimistic and wholesome
4. **Respectful of Time**: Offline progress ensures you're never punished for taking breaks
5. **Character-Driven**: Everything ties back to Milo's personality and interests

## Save System

- The game auto-saves every 30 seconds
- Saves are stored in `milo_save.json` in the game directory
- Offline progress is calculated when you return (max 24 hours)
- To reset your save, simply delete `milo_save.json`
