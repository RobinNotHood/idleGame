Subtitle Adjust

Adjust the position and size of subtitles in the Jellyfin web client. After saving, reload the web client for changes to take effect.

Note: This plugin exposes a script at /SubtitleAdjust/ClientScript.js. Load it from your web client (for example by appending <script src="/SubtitleAdjust/ClientScript.js"></script> to jellyfin-web/index.html, or via a custom-CSS/JS plugin) to apply these settings on the client.

Uncheck to disable all style overrides without uninstalling the plugin.
100 = Jellyfin default. 150 makes subtitles 1.5x larger; 75 makes them smaller.
Distance of subtitles from the bottom of the video, in percent of the viewport height. Higher values push subtitles further up.
Shift subtitles left (negative) or right (positive) from the center, in percent of the viewport width. 0 keeps them centered.
Any valid CSS color (e.g. #FFFFFF, yellow, rgb(255, 255, 0)).
Background behind the subtitle text. Use transparent to disable the background.
CSS font-family stack. Leave as inherit to use the Jellyfin default.
Adds a subtle dark shadow behind each character so subtitles stay legible against bright backgrounds.
