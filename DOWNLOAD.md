# Download Milo's Cozy Adventures

## Quick Start - Play Now!

### Option 1: Download Executable (Easiest)
1. Go to the [dist folder](https://github.com/RobinNotHood/idleGame/tree/main/dist)
2. Click on `MilosCozyAdventures.exe`
3. Click the "Download" button
4. Run the .exe file - no installation needed!

**Direct Download Link:**
[MilosCozyAdventures.exe](https://github.com/RobinNotHood/idleGame/raw/main/dist/MilosCozyAdventures.exe) (17 MB)

### Option 2: Run from Source
```bash
# Clone the repository
git clone https://github.com/RobinNotHood/idleGame.git
cd idleGame

# Install dependencies
pip install -r requirements.txt

# Run the game
python main.py
```

## What You Get

✨ **Full Game Features:**
- Dark mode (press 'D' to toggle)
- Resizable window
- Offline progress (up to 24 hours)
- Auto-save every 30 seconds
- 17+ upgrades
- 8 unique adventures
- Modern, polished UI

## System Requirements

- Windows 10/11 (64-bit)
- 256 MB RAM
- 50 MB disk space
- Display: 1280x720 recommended (minimum 800x600)

## Controls

- **Left Click**: Interact with Milo and buttons
- **D Key**: Toggle dark mode
- **Resize**: Drag window edges
